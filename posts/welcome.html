Welcome to Sypher!

A very simple minimalist static blog system.

Sypher is a very simple static blog system, minimalist and lightweight. It's intended to allows you write fast using an easy-to-use language and a script generates all HTML and CSS files needed to it working right. See below some advantages.

1 - It can be used in GitHub Pages.

Github pages is a hosting service. You can host only a static website and Sypher was built focusing it, obviously you can host a Sypher blog in any place, but using GitHub pages you can do it for free and fast!

2 - Use a VERY simple Markdown language to write posts.

Markdown it's a very simple markup language that converts text into HTML. You do write it in a plain text file(using a Markdown format) and it will be converted in runtime to a pretty blog post.

3 - Sypher is design responsive.

You can read your posts with any device, it works well since 480px width devices.

4- It's free to use.

Sypher is free to use, just fork the github repo and start to write!

5- There aren't external scripts needed and no technical knowledge required.

Sypher is a web front-end working in runtime, it means you don't need to install and learn how to use nodeJS, Ruby or any additional tool to be able to use that. It works only in a web browser as Firefox or Chrome. Using Sypher you need to worry just about your post content and nothing more.
